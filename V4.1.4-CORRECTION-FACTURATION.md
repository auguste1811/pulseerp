# Correction urgente factures et devis

Appliquez d'abord les migrations :

```bash
npx prisma migrate deploy
npx prisma generate
```

Puis vérifiez :

```bash
npm run verify
```

La migration `20260723120000_invoice_email_log` crée la table
`sales_document_emails`. La version 4.1.4 reste néanmoins fonctionnelle
même si cette migration n'est pas encore appliquée.

Pour le partage SMS/WhatsApp, ajoutez dans Vercel :

```env
INVOICE_SHARE_SECRET=une-longue-valeur-aleatoire
```
