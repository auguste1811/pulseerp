# PulseERP v1.0.1 — Landing page publique

## Ajouts

- Page d'accueil publique sur `/`
- Navigation marketing
- Bouton Connexion vers `/login`
- Présentation des principaux modules
- Aperçu visuel de l'interface
- Présentation du centre d'automatisation
- Appels à l'action
- Responsive mobile et tablette
- Styles isolés dans un CSS Module

## Installation

Copier le contenu du ZIP sur le projet actuel, puis :

```bash
rm -rf .next
npm run dev
```

La page publique est disponible sur :

```text
http://localhost:3000
```
