# PulseERP v0.8.0 — Paramètres entreprise

## Nouveautés

- Page Paramètres professionnelle
- Coordonnées légales de l'entreprise
- Adresse, SIRET et numéro de TVA
- Email, téléphone et site internet
- IBAN et BIC
- Devise et TVA par défaut
- Délai de paiement par défaut
- Durée de validité des devis
- Préfixes personnalisés des devis et factures
- Mentions légales en pied de document
- Modification du profil utilisateur
- Changement de mot de passe
- Affichage automatique des coordonnées sur les PDF
- Numérotation utilisant les préfixes configurés
- Échéances automatiques selon les paramètres

## Installation

```bash
npm run setup
rm -rf .next
npm run dev
```

## Accès

```text
http://localhost:3000/settings
```
