# PulseERP v3.1 — Facturation & Comptabilité

## Fonctionnalités

- Factures de vente visibles automatiquement en comptabilité
- Création/mise à jour automatique du mouvement financier lié
- Import PDF/JPG/PNG des factures d’achat (8 Mo maximum)
- Stockage sécurisé du justificatif dans PostgreSQL/Neon
- Classement par catégorie
- Statut à payer, payé ou en retard
- Écritures d’achat automatiques : charge, TVA déductible, fournisseur
- TVA collectée, TVA récupérable et résultat estimé
- Consultation sécurisée des justificatifs

## Installation

```bash
npm install
npx prisma migrate deploy
npx prisma generate
npm run build
npm run dev
```

Puis :

```bash
git add .
git commit -m "feat: facturation et comptabilité v3.1"
git push origin main
```

Vercel appliquera le nouveau build. La migration doit être appliquée une fois
à Neon depuis le terminal local avec `npx prisma migrate deploy`.

## Limite de cette version

La lecture OCR automatique n’est pas activée : l’utilisateur importe la pièce
et renseigne les montants. La structure est prête pour ajouter un moteur OCR
ou une API d’extraction dans une version suivante.
