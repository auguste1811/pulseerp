# PulseERP v3.7.0 — Super Admin & Modules

## Fonctionnalités

- Back-office global `/admin`
- Création d’une entreprise cliente
- Création automatique du compte propriétaire
- Mot de passe temporaire
- Activation module par module
- Suspension et réactivation
- Date d’expiration facultative
- Menu dynamique pour les clients
- Protection serveur des routes
- Conservation de tous les modules pour les entreprises existantes
- Aucun changement Stripe
- Aucun secret modifié

## Activation de votre compte super administrateur

### Méthode recommandée

Ajoutez dans `.env` et Vercel :

```env
PLATFORM_ADMIN_EMAILS=votre@email.fr
```

Plusieurs emails peuvent être séparés par des virgules.

### Méthode base de données

```bash
npm run platform:promote -- votre@email.fr
```

## Installation

```bash
npm install
npx prisma migrate deploy
npx prisma generate
npm run build
```

Puis :

```bash
git add .
git commit -m "feat: super admin et modules v3.7"
git push origin main
```

## Parcours

1. Connectez-vous avec votre compte développeur.
2. Ouvrez `/admin`.
3. Cliquez sur `Nouvelle entreprise`.
4. Saisissez le responsable et un mot de passe temporaire.
5. Cochez les outils demandés.
6. Transmettez les identifiants au client.
7. Modifiez les modules à tout moment depuis sa fiche.

Les outils désactivés sont à la fois masqués dans le menu et bloqués côté serveur.
