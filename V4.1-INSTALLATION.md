# PulseERP v4.1.0 — Installation

## Corrections incluses

- Graphique réel du tableau de bord sur 12 mois.
- Rapport du chiffre d'affaires plus lisible : revenus, dépenses, axes et détail par période.
- Recherche globale fonctionnelle depuis la barre supérieure (`Cmd/Ctrl + K`).
- Groupes de contacts : création, suppression, filtre, import CSV, affectation multiple et pipeline.
- Création de plusieurs entreprises avec un seul compte.
- Sélecteur d'entreprise active.
- Correction du schéma Prisma du CRM Téléphone.

## Installation

```bash
rm -rf .next
npm install
npx prisma migrate deploy
npx prisma generate
npm run verify
```

## Déploiement

```bash
git add .
git commit -m "feat: graphiques recherche groupes contacts et multi-entreprises v4.1"
git push origin main
npx vercel --prod --force
```

## Tests rapides

1. Ajouter une opération de revenu et vérifier le graphique du dashboard.
2. Ouvrir Rapports et changer 7 jours / 30 jours / Année.
3. Rechercher un contact, une facture ou une tâche dans la barre supérieure.
4. Créer un groupe de contacts.
5. Importer un CSV en choisissant ce groupe.
6. Affecter plusieurs groupes depuis une fiche contact.
7. Filtrer la liste et le pipeline par groupe.
8. Créer une seconde entreprise depuis Mes entreprises.
9. Basculer entre les entreprises et vérifier l'isolation des données.
