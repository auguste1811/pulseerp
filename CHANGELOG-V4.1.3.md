# PulseERP v4.1.3

- Envoi des factures par SMS avec message prérempli.
- Envoi des factures par WhatsApp avec message prérempli.
- Lien PDF public sécurisé par signature HMAC.
- Numéro du client repris automatiquement depuis le CRM.
- Message personnalisable avant l’envoi.
- Bouton de copie du message.
- Aucune API SMS ou WhatsApp payante obligatoire.
- Le client peut consulter le PDF sans se connecter à PulseERP.
- Les liens sont protégés contre la modification et l’indexation.
