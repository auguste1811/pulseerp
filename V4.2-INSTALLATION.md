# Installation PulseERP v4.2.0

```bash
rm -rf .next node_modules
npm install
npx prisma migrate deploy
npx prisma generate
npm run verify
```

Variable nécessaire au partage des factures :

```env
INVOICE_SHARE_SECRET=une-valeur-aleatoire-longue
```

Les variables `RESEND_API_KEY` et `EMAIL_FROM` peuvent être supprimées de Vercel si elles ne servent pas à une autre fonction.
