# Factures par SMS et WhatsApp

## Fonctionnement

Depuis une facture :

1. Ouvrez la fiche de la facture.
2. Descendez jusqu’à `Envoyer par message`.
3. Modifiez éventuellement le texte.
4. Cliquez sur `Envoyer par SMS` ou `Envoyer par WhatsApp`.
5. L’application Messages ou WhatsApp s’ouvre avec le numéro, le texte et le lien PDF préremplis.
6. Confirmez l’envoi depuis votre téléphone ou ordinateur.

## Configuration

La fonctionnalité fonctionne avec `AUTH_SECRET`. Pour utiliser un secret séparé :

```env
INVOICE_SHARE_SECRET=une-valeur-longue-et-aleatoire
```

Aucune clé Twilio ou Meta n’est requise dans cette version.

## Limite

PulseERP prépare le message et ouvre l’application choisie. L’envoi final est confirmé par l’utilisateur. Cela permet de rester gratuit et d’éviter une API de messagerie payante.
