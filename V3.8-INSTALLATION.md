# PulseERP v3.8 — Installation

```bash
rm -rf .next
npm install
npx prisma migrate deploy
npx prisma generate
npm run build
```

Puis :

```bash
git add .
git commit -m "feat: acquisition, rendez-vous CRM et revenus manuels v3.8"
git push origin main
```

## Tests

1. Créer un contact avec la source Ads.
2. Ouvrir `/acquisition` et vérifier que le lead est compté.
3. Renseigner les dépenses Ads et vérifier le coût par lead.
4. Attribuer le canal à un utilisateur.
5. Ouvrir une fiche contact et créer un prochain rendez-vous.
6. Vérifier sa présence dans `/calendar`.
7. Ajouter un revenu manuel depuis `/dashboard`.
8. Vérifier sa présence dans `/transactions`.
