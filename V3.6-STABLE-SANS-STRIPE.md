# PulseERP v3.6.0 — Stabilisation sans Stripe

Cette version neutralise Stripe Billing et Stripe Connect dans l’interface afin
qu’aucune erreur de paiement ne bloque l’application.

## Comportement

- accès permanent à l’application ;
- aucun blocage après trois jours ;
- aucune redirection automatique vers Stripe ;
- page `/subscribe` remplacée par une page Preview gratuite ;
- boutons Stripe affichés comme « Bientôt disponible » ;
- aucune variable d’environnement modifiée ou requise pour Stripe ;
- Auth.js, Neon, CRM, facturation, comptabilité, calendrier et tâches conservés.

## Installation

```bash
rm -rf .next
npm install
npx prisma migrate deploy
npx prisma generate
npm run build
```
