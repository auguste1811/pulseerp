# Correction de l’erreur après inscription

## Installation

```bash
npm install
npx prisma migrate deploy
npx prisma generate
npm run build
git add .
git commit -m "fix: inscription et diagnostic Auth v3.5.2"
git push origin main
```

## Diagnostic

Après le nouveau déploiement, ouvrir :

```text
https://VOTRE-DOMAINE/api/health/config
```

Résultat attendu :

```json
{
  "ok": true,
  "checks": {
    "authSecretConfigured": true,
    "databaseUrlConfigured": true,
    "databaseReachable": true,
    "subscriptionSchemaReady": true
  }
}
```

Le diagnostic ne renvoie jamais les valeurs des secrets.
