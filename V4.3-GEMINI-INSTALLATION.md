# PulseERP v4.3 — Installation Gemini

## Variables Vercel

```env
GEMINI_API_KEY=votre_cle_google_ai_studio
GEMINI_MODEL=gemini-2.5-flash
GEMINI_DAILY_LIMIT=50
```

Ne préfixez jamais la clé avec `NEXT_PUBLIC_`.

## Installation

```bash
rm -rf .next node_modules
npm install
npx prisma migrate deploy
npx prisma generate
npm run verify
```

## Déploiement

```bash
git add .
git commit -m "feat: PulseAI Gemini v4.3"
git push origin main
npx vercel --prod --force
```

## Test

1. Ouvrir `/api/health/config` et vérifier `geminiConfigured: true`.
2. Ouvrir `/ai`.
3. Demander : `Résume mes priorités de la semaine.`
4. Vérifier que la conversation apparaît dans l’historique.
