# PulseERP v1.0.0 Premium

ERP SaaS web pour TPE et PME.

## Démarrage

```bash
cp .env.example .env
docker compose up -d db
npm install
npm run setup
npm run dev
```

Ouvrir `http://localhost:3000`.
