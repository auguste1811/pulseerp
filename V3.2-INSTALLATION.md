# PulseERP v3.2.0 — PulseAI

## Configuration

Ajoutez localement et dans Vercel :

```env
OPENAI_API_KEY=votre_cle_api
OPENAI_MODEL=gpt-4.1-mini
```

La clé doit rester exclusivement côté serveur. Ne la publiez jamais sur GitHub.

## Installation

```bash
npm install
npx prisma migrate deploy
npx prisma generate
npm run build
```

## Fonctions

- Assistant contextuel sur `/ai`
- Analyse CA, dépenses, TVA, factures, CRM et tâches
- Historique des conversations par entreprise
- Analyse IA des factures d’achat PDF/JPG/PNG
- Mise à jour automatique de la facture, de la dépense et des écritures comptables

L’analyse comptable reste une aide et ne remplace pas la validation d’un expert-comptable.
