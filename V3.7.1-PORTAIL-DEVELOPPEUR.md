# PulseERP v3.7.1 — Portail développeur

## Création du compte développeur

Après avoir appliqué les migrations Prisma :

```bash
npm run platform:create-admin -- \
  votre-email@exemple.fr \
  "VotreMotDePasseSolide!" \
  Auguste \
  Martinat
```

Le compte créé :

- est administrateur global ;
- n’a pas besoin d’appartenir à une entreprise ;
- se connecte sur `/developer/login` ;
- accède directement à `/admin`.

## Connexion

```text
https://votre-domaine.vercel.app/developer/login
```

Utilisez l’email et le mot de passe passés à la commande.

## Installation

```bash
npm install
npx prisma migrate deploy
npx prisma generate
npm run platform:create-admin -- email mot-de-passe prénom nom
npm run build
```

## Sécurité

- aucune entreprise cliente ne peut accéder au portail ;
- une URL `/admin` non autorisée renvoie une page introuvable ;
- le compte plateforme peut exister sans entreprise ;
- les comptes clients continuent d’utiliser `/login`.
